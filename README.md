# FDOT-Reconstruction

This repository demonstrates forward simulation and inverse reconstruction for fluorescence diffuse optical tomography (FDOT) using synthetic measurement data.

## Overview

The project focuses on reconstructing target-related parameters from simulated FDOT signals generated by a forward model.  
The implementation includes forward measurement generation, temporal signal processing, and optimization-based reconstruction.

## Pipeline

Forward model  
→ Synthetic measurement generation  
→ Signal normalization / temporal processing  
→ Parameter optimization  
→ Target reconstruction

## Files

- `FDOT_github.ipynb` : notebook for running the reconstruction pipeline
- `FDOT_func.py` : core functions for forward simulation, signal processing, and reconstruction
- `requirements.txt` : required Python packages

## Features

- FDOT forward simulation
- Synthetic measurement generation
- Optimization-based inverse reconstruction
- Training history saving
- Reconstruction result visualization

## Notes

- The repository uses synthetic data generated from the forward model.
- This project is intended as a research/demo implementation.
- Some experiment-specific settings may need to be adjusted depending on the reconstruction setup.
